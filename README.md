# fastswoole

#### 介绍
基于swoole写的一个MVC框架,功能正在逐步完善

#### 主要特性

基于swoole写的一个MVC框架,目前已实现
* 控制反转容器DI
* APP流程控制
* 控制器
* ORM模型类
* 异常类
* URL生成类
* 视图类


## 安装

~~~
composer create-project alex-lee/fastswoole fastswoole
~~~

如果需要更新框架使用
~~~
composer update alex-lee/fastswoole
~~~
